# Gatherly

Gatherly is a community-driven platform that helps people discover, create, and share local events.
It makes it easy for anyone to browse nearby happenings, list their own events, and connect with others through simple, intuitive toolsâ€”all in one place.

## Features
  - RSVP to events
  - Browse events
  - Add your own events

## Tech Stack
  - Next.js
  - Tailwind CSS
  - Prisma ORM
  - PostgreSQL

## Installation/Cloning

```bash
git clone https://github.com/flappibird123/gatherly.git
cd gatherly
```
Choose what version you want to run
e.g. 
```bash
cd v0.2
npm install
```

## Usage/Running
```bash
npm run dev
```
The app should start by default on `http://localhost:3000`

## Configuration
Change the  `DATABASE_URL` in the `.env` file to match the database you are using
